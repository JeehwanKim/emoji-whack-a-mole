# Emoji Whack-a-Mole

A lightweight web-based 3x3 whack-a-mole game using emojis. The game runs
for 30 seconds and speeds up as time progresses.

## Run

Open `index.html` in your browser.

## Files

- `index.html` - Markup for the game UI.
- `styles.css` - Game styling.
- `app.js` - Game logic (spawning, timing, scoring).

## Customize

- Change emojis in `app.js` under `emojis`.
- Tweak game speed with `BASE_INTERVAL` and `MIN_INTERVAL`.
- Adjust duration with `GAME_DURATION`.
